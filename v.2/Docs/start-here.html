Installing The Template

  1. After unzip Theme Files, you will found HTML Folder all files inside you will upload them to your site.
  2. You can view this Theme in any web browser from your desktop computer. Because the files are written in HTML, you do not need an internet connection in order to display or edit the Theme.
  3. Start your FTP client (I persoanlly use Filezila which is free) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.
  4. Locate the domain folder you wish to upload your Theme files to on your server. You are going to want to click and drag every file that you have labeled .html, assets folder into your FTP client.
  5. Once the files are done uploading, rename the file you want to be your home page as 'index.html'.

CSS Structure

This template powered by sass, it consists of many .scss files that compiled and called from 1 file called import so all these files will be called from one file for fast page loading but in fact you can manage or edit them separately.

You can find all scss files inside this path assets/scss

Check import.scss :


/* Fonts */
@import url('https://fonts.googleapis.com/css?family=Montserrat:400,700|Roboto:400,700');

/* Bootstrap */
@import "../node_modules/bootstrap/scss/bootstrap";

/* Animate css */
@import "../node_modules/animate.css/animate.min.css";
